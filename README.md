# LinearRegressionModel
Machine Learning project to predict Amazon stock prices using Linear Regression based on historical market data, with performance evaluation and visualization
# 📊 Amazon Stock Price Prediction

## 📌 Description
This project focuses on predicting stock prices of Amazon using Machine Learning. Historical stock data is collected using the yfinance library and processed using pandas and NumPy.

A Linear Regression model from scikit-learn is used to predict the closing price based on features like Open, High, Low, and Volume.

## 🔧 Technologies Used
- Python
- pandas
- NumPy
- scikit-learn
- Matplotlib
- yfinance

## 📈 Machine Learning Model
- Linear Regression

## 📊 Features Used
- Open
- High
- Low
- Volume

## 🎯 Target Variable
- Close Price

## ⚙️ Workflow
1. Data collection using yfinance  
2. Data preprocessing using pandas  
3. Splitting dataset into training and testing sets  
4. Training Linear Regression model  
5. Making predictions  
6. Evaluating model performance  
7. Visualizing results  

## 📉 Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score

## 📊 Visualization
- Scatter plot of Actual vs Predicted prices
